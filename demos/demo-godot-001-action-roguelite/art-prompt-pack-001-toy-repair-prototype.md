# Toy Repair Prototype - Image Prompt Pack

用于直接复制到 AI 图像工具生成第一批玩具修理屋 Prototype 资源。

## 通用风格前缀

中文：

```text
童话治愈风格，夜晚玩具修理屋氛围，温暖台灯光，手绘绘本质感，木头、布料、发条、纽扣、螺丝、小工具元素，可爱但不幼稚，清晰轮廓，高对比，适合俯视角2D动作游戏
```

English:

```text
cozy fairytale style, nighttime toy repair shop mood, warm desk lamp lighting, hand-painted storybook texture, wood, fabric, wind-up keys, buttons, screws, small tool elements, cute but not childish, clear readable silhouette, high contrast, suitable for top-down 2D action game
```

## 通用 Negative Prompt

```text
realistic photo, horror, creepy doll, blood, gore, realistic weapon, cyberpunk, sci-fi armor, complex background, text, logo, watermark, low contrast, too many tiny details, blurry, noisy
```

## 选择标准

生成后优先挑选满足这些条件的版本：

- 小图缩到 64x64 仍能看清轮廓。
- 玩家、近战敌人、远程敌人颜色和轮廓能区分。
- 背景中央区域不抢眼。
- 风格温暖，不恐怖。
- 玩具/修理元素明确。
