# Spec 025：P1 硬阻挡区 0.1

## 背景

当前房间已有少量桌面障碍和胶水坑，但硬阻挡区的主题表达仍偏弱。P1 需要加入更像玩具修理屋场景的硬阻挡：工具盒、木块堆、玩具箱边缘。

## 目标

- 每个房间增加更明确的硬阻挡区主题。
- 硬阻挡继续阻挡玩家、敌人、投射物。
- 导航区域继续从硬阻挡中挖洞，敌人能绕行。
- 不引入外部图片资源，先用代码绘制多层 ColorRect 表现。

## 方案

- 扩展 `room_maps.obstacles` 数据：增加 `kind` 字段。
  - `toolbox`：工具盒。
  - `woodpile`：木块堆。
  - `toybox_edge`：玩具箱边缘。
  - `block`：原普通桌面块。
- `_create_obstacle()` 根据 `kind` 增加无资源装饰：
  - 工具盒：暗红/棕色主体 + 金属扣。
  - 木块堆：主体 + 多条木块分隔线。
  - 玩具箱边缘：长条边框 + 高光线。
- 保持碰撞体和导航洞使用同一 `size`。

## 验证

- smoke test 通过。
- 手测验证硬阻挡视觉更明显、玩家/敌人/投射物不可穿越、敌人能绕行。
