# Spec 020：地图边缘视觉提示 0.1

## 背景

当前玩家移动边界已收缩，角色不会贴到地图最外边缘。但画面上缺少“这里不能走”的视觉提示，玩家可能误以为桌面边缘或远处背景仍可跨越。

本轮用无新资源的方式先做边缘视觉提示，后续可替换为玩具箱边框、桌布边、工具堆等美术资源。

## 本轮目标

1. 在可移动区域边界外增加低透明暗边/框线提示。
2. 不影响碰撞、寻路和相机。
3. 风格保持克制，不遮挡桌面背景。
4. 后续可被正式边缘美术替换。

## 非目标

- 不生成新图片资源。
- 不改变玩家可移动范围。
- 不新增新的硬碰撞体。
- 不重排障碍。

## 实现方案

在 `MapRoot` 下增加一个视觉专用容器：

```text
BoundaryHint
```

运行时创建 4 条 `ColorRect`：

- Top 暗边
- Bottom 暗边
- Left 暗边
- Right 暗边

边界基于玩家安全移动区域：

```text
play_area_min = Vector2(80, 120)
play_area_max = Vector2(3088, 1264)
```

表现：

- 黑棕色低透明遮罩：`Color(0.05, 0.025, 0.01, 0.24)`
- 内侧细描边：暖木色 `Color(0.8, 0.55, 0.28, 0.36)`

## 验收标准

1. 运行后边缘区域有轻微暗边，提示不可走。
2. 不遮挡主战斗区域。
3. 不影响障碍、胶水坑、敌人导航。
4. `tests/smoke_test.gd` 输出 `AI_TEST_PASS`。
