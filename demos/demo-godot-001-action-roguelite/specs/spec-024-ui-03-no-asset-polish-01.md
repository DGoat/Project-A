# Spec 024：UI 0.3 无资源打磨

## 背景

P0 必要项已完成，只剩可选 UI 图片资源。当前先不引入新图片，用 Godot 控件和样式完成 UI 0.3 打磨，降低资源依赖。

## 目标

- 开始界面更像“玩具修理屋”的入口，不只是一块文本面板。
- 三选一更像卡牌，有标题、正文、快捷键与悬停层级。
- HUD 更轻、更稳定，保留战斗可读性。
- 结算界面与开始界面风格统一。

## 方案

1. 增加 UI 代码样式层：
   - 继续使用 `StyleBoxFlat`。
   - 增加高光、阴影、圆角、边框差异。
   - 按面板类型区分：主面板、卡牌、HUD、调试面板。

2. 增加无资源装饰：
   - 开始界面加入上下分隔线/小装饰文本。
   - 三选一卡牌增加编号和 tag 层级。
   - HUD 保持小胶囊，但微调对比度和边距。

3. 保持功能不变：
   - 不修改输入、战斗、祝福逻辑。
   - 不增加外部图片资源。

## 验证

- smoke test 通过。
- 手测检查开始界面、三选一、HUD、结算界面是否更统一。
